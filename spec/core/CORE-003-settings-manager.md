# CORE-003 - Settings Manager

> **Module**: Core  
> **Requirement**: CORE-016 through CORE-020  
> **Status**: To Do

## 1. Overview

Settings Manager handles all player preferences including graphics, audio, and controls.

## 2. Requirements

| ID | Requirement | Description |
|----|-------------|-------------|
| CORE-016 | Graphics Settings | Resolution, quality, vsync |
| CORE-017 | Audio Settings | Volume, music, sfx |
| CORE-018 | Control Settings | Key bindings |
| CORE-019 | Gameplay Settings | Difficulty, UI scale |
| CORE-020 | Settings Persistence | Save/load settings |

## 3. Acceptance Criteria

- [ ] All settings can be modified
- [ ] Settings persist across sessions
- [ ] Default settings work on first run
